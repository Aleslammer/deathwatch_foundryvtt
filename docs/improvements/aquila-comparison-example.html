Detailed Comparison

Unicode Emoji 🦅

✅ Pros
  • No external files required
  • Zero file size overhead
  • Works offline immediately
  • Easy to implement (content: '🦅';)
❌ Cons
  • Platform-dependent rendering — looks different on Windows/Mac/Linux/mobile
  • Not the Imperial Aquila — generic eagle design
  • Limited styling — can't change color, add glow, or filter
  • Cartoonish appearance — doesn't match 40K aesthetic
  • Size limitations — may look blurry when scaled
  • Inconsistent with system — other icons are authentic

Authentic Icon

✅ Pros
  • Official Warhammer 40K design — double-headed eagle, proper proportions
  • Consistent rendering — looks identical on all platforms
  • Full CSS control — color, filters, shadows, transforms
  • Professional quality — high detail, scalable
  • Already in your codebase — src/icons/aquila.webp
  • Thematically authentic — instant 40K recognition
  • Matches system aesthetic — consistent with other icons
❌ Cons
  • Optimized WebP format (very small file size)
  • Requires image file in correct location
  • Slightly more complex CSS

CSS Implementation

Unicode Emoji Approach

.sheet-header::before {
  content: '🦅';
  position: absolute;
  top: -5px;
  left: -5px;
  font-size: 48px;
  opacity: 0.15;
  transform: rotate(-15deg);
}

Authentic Icon Approach (Recommended)

.sheet-header::before {
  content: '';  /* Empty — using background image */
  position: absolute;
  top: -10px;
  left: -10px;
  width: 70px;
  height: 70px;
  background: url('../icons/aquila.webp') no-repeat center;
  background-size: contain;
  opacity: 0.18;
  transform: rotate(-12deg);
  filter: drop-shadow(0 0 8px rgba(212, 165, 116, 0.4));
}
Key Differences:
• content: '' (empty) instead of content: '🦅'
• background: url(...) to load the image
• width and height to size the icon
• filter: drop-shadow(...) for glow effect (optional)

RECOMMENDATION

✅ Use the Authentic Aquila Icon

Your system already has src/icons/aquila.webp — an authentic, high-quality Imperial Aquila icon. Use it instead of Unicode emojis for:

  • Sheet header corners

Result: Professional, authentic Warhammer 40K interface that looks identical on all platforms and instantly communicates the Imperial aesthetic. The optimized WebP file size is very small (cached after first load) and the implementation is only slightly more complex than Unicode emojis.
