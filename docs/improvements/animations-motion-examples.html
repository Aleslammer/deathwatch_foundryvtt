⚡ Performance Guidelines

✅ GPU-Accelerated

Use these (fast):
• transform (translate, scale, rotate)
• opacity
• filter (brightness, blur)

These properties are hardware-accelerated and perform smoothly even on lower-end systems.

❌ Layout-Triggering

Avoid these (slow):
• width, height
• top, left (use transform instead)
• background-position

These force layout recalculation and can cause jank.

⚙️ Animation Duration Guidelines

Instant feedback 150ms Button press, checkbox toggle
Quick transition 250-300ms Hover effects, color changes
Standard animation 400-500ms Scan lines, tab switches
Deliberate effect 600-800ms Dialog entrance, value flash
Ambient loop 2-5s Pulse effects, scan sweeps

KEY TAKEAWAYS

✓ Strategic motion — High-impact moments, not scattered micro-interactions

✓ GPU-accelerated — Use transform and opacity for smooth 60fps animations

✓ Thematic consistency — HUD scans, tactical highlights, power armor activation

✓ Tactile feedback — Press states, hover glows, pulse warnings

✓ Deliberate timing — Ancient tech feels weighty, not snappy modern UI
