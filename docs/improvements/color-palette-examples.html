Color Usage Guidelines

Color Use For
Imperial Gold Headers, links, primary actions, borders, active states
Deathwatch Silver Text, borders, secondary elements, ceramite surfaces
HUD Green Rollable hover states, tactical displays, success messages
HUD Amber Warnings, sustained abilities, caution indicators
Critical Red Critical wounds, severe failures, danger states
Plasma Glow Tech effects, weapon glows, energy indicators
Chapter Red Blood Angels chapter accents, dramatic emphasis

IMPLEMENTATION

Step 1: Replace CSS variables in src/styles/variables.css

Step 2: Update component references (buttons, links, headers)

Step 3: Test all UI elements for contrast and readability

Result: Interface transforms from generic Bootstrap to unmistakably Warhammer 40K Imperial aesthetic.
