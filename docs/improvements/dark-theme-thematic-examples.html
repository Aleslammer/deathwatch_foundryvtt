IMPLEMENTATION SUMMARY

Dark Theme: War Room atmospheric backgrounds with multi-layer gradients

Thematic Icons: Aquila, skulls, seals, rivets — all via CSS pseudo-elements

Decorative Elements: Corner brackets, wax seals, watermarks at low opacity

Visual Hierarchy: Icons reinforce meaning (medica cross on wounds, skull on fatigue)

Performance: All CSS-based, no external images required
