Implementation Summary

Font Usage Guidelines
Cinzel → Headers (h1, h2, h3), Section headers, Tab labels, Character names

Rajdhani → Body text, Item names, Skill names, Descriptions, Labels

Share Tech Mono → Characteristic values, Stat numbers, Damage rolls, Numeric displays
Why This Works
✓ Cinzel = Imperial Gothic architecture, High Gothic texts, imposing and reverent
✓ Rajdhani = Military technical readouts, slightly futuristic, highly readable
✓ Share Tech Mono = Power Armor HUD displays, tactical data, monospace precision

Combined, they create a distinctive Warhammer 40K aesthetic that feels ancient, militaristic, and battle-ready. The fonts work together to establish visual hierarchy and thematic immersion.
