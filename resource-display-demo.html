Design Features

  • Angled clip-path borders — Evokes armor plating and tactical displays
  • Corner accent brackets — Imperial gothic framing elements
  • Color-coded panels — Amber (XP), Cyan (Fate), Gold (Renown)
  • Scan-line animation on hover — Active auspex scanning effect
  • Glowing hover states — Gradient border glow on interaction
  • Monospace data fields — Tactical readout aesthetic
  • Text-shadow glows — HUD-style illumination matching panel type
  • Status indicators — Green (positive), Red pulsing (critical/negative)
  • Input focus effects — Gold glow when editing values
